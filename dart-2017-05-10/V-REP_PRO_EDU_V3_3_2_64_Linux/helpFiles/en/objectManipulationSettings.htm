Object Manipulation Settings

The object manipulation settings can be displayed with [Menu bar --> Tools --> Object manipulation settings]. The content of the dialog is dependent on the last selected object. If no object is selected, the dialog is inactive.



[Mouse translation/rotation settings dialog]


Mouse translations

  • Relative to world/parent frame/own frame: indicates that a mouse drag will translate the selected object on a plane or line that is aligned with the absolute reference frame, aligned with the parent object reference frame, or aligned with the object's own reference frame.
  • Preferred axes: along X/ along Y/ along Z: indicates that a mouse drag allows translating the selected object along preferred axes of the reference frame selected above. Other axes can be used during manipulation by pressing the ctrl-key after the mouse button was pressed down.
  • Translation step size: the step size used when translating the selected object with a mouse drag (see the user settings dialog for default step sizes). Smaller step sized can still be used during manipulation by pressing the shift-key after the mouse button was pressed down.


    • Mouse rotations

  • Relative to world/parent frame/own frame: indicates that a mouse drag will rotate the selected object about an axis of the absolute reference frame, the parent object reference frame, or the object's own reference frame.
  • Preferred axis: about X/ about Y/ about Z: indicates that a mouse drag allows rotating the selected object about a preferred axis of the reference frame selected above. Other axes can be used during manipulation by pressing the ctrl-key after the mouse button was pressed down.
  • Rotation step size: the step size used when rotating the selected object with a mouse drag (see the user settings dialog for default step sizes). Smaller step sized can still be used during manipulation by pressing the shift-key after the mouse button was pressed down.


  • Mouse manipulation disabled when simulation running / not running: when enabled, then the selected object cannot be manipulated when the simulation is running / not running.
