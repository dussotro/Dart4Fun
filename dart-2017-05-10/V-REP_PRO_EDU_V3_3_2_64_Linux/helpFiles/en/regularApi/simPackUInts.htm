Regular API function

simPackUInts

Description Packs a table of unsigned integer numbers into a string. See also simUnpackUInts and the other packing/unpacking functions.
C synopsis -
C parameters
-
C return value
-
Lua synopsis string data=simPackUInts(table unsignedIntegerNumbers,number startUIntIndex=0,number uIntCount=0)
Lua parameters
unsignedIntegerNumbers: a table containing unsigned integer numbers. Non-numbers will be packed as zero values
startUIntIndex: the zero-based index from which on data should be packed. Can be omitted in which case 0 is used
uIntCount: the amount of uints that should be packed. Can be omitted in which case 0 is used (which indicates that the maximum available uints should be packed from the indicated startUIntIndex)
Lua return values
data: a string (values between 0 and 255) that contains packed unsigned integer numbers from the table if operation was successful, nil otherwise
