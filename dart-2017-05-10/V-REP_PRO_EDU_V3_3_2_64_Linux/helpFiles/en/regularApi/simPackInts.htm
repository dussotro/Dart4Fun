Regular API function

simPackInts

Description Packs a table of integer numbers into a string. See also simUnpackInts and the other packing/unpacking functions.
C synopsis -
C parameters
-
C return value
-
Lua synopsis string data=simPackInts(table integerNumbers,number startIntIndex=0,number intCount=0)
Lua parameters
integerNumbers: a table containing integer numbers. Non-numbers will be packed as zero values
startIntIndex: the zero-based index from which on data should be packed. Can be omitted in which case 0 is used
intCount: the amount of ints that should be packed. Can be omitted in which case 0 is used (which indicates that the maximum available ints should be packed from the indicated startIntIndex)
Lua return values
data: a string (values between 0 and 255) that contains packed integer numbers from the table if operation was successful, nil otherwise
